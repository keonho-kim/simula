# senario.samples

## 구성

- `01_i-am-solo_31_2026-04-10.md`
  - 연애 관계 재편과 최종 선택
- `02_wargame_iran_us.md`
  - 제한전 확전과 전쟁 종결 경로
- `03_startup_boardroom_crisis.md`
  - 이사회 권력 충돌과 생존 결정
- `04_city_hall_disaster_response.md`
  - 재난 대응과 초기 안정화
- `05_campus_election_scandal.md`
  - 선거 정당성과 딥페이크 위기
- `06_fantasy_court_intrigue.md`
  - 계승 분쟁과 전선 통제

## 작성 원칙

- 모든 샘플은 `상황 시작`부터 `최종 판정 또는 종료 시점`까지를 목표로 둔다.
- `나는 SOLO`류는 `최종 선택`까지, `war game`류는 `휴전·종전·확전 구조 확정`까지 본다.
- 세부 장면을 과도하게 늘어놓기보다, actor·압력·전환점·종료 조건만 남기는 쪽을 우선한다.
- 일부 시나리오는 현실 보도나 현실 국가를 참고해 재구성했으므로 실제 전개와 다를 수 있다.
